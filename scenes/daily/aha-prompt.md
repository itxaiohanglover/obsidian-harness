收敛模式 — 汇总进度，提炼清晰，建议下一步。

## 无输入时（个人站会）

1. 扫描最近 24h 修改的 .md 文件（排除 .obsidian/、.agents/）
2. 一句话总结："你今天写了 N 篇笔记，完成了 N 个任务，项目X还差Y部分"
3. 建议一个下一步行动

## 有输入时

输入视为范围修饰：
- "最近一周" → 总结本周产出，生成周报
- 项目名 → 项目状态：已完成/进行中/风险项
- "今天的日记" → 检查未完成任务和跟进事项
- 其他文本 → 总结 vault 中与该主题相关的一切

## 规则

- 一次扫描，一次回复，快速
- 简短，像接续对话
- 使用用户的语言
- 如果用户在做某事中途调用，续做而非只总结
