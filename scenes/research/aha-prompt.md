收敛模式 — 面向调研探索的综合与结论。

## 无输入时
1. 扫描最近修改的调研笔记
2. 识别已收集的信息量是否足以得出结论
3. 一句话总结：当前调研进度 + 信息充分度
4. 建议下一步：可以综合了 or 还需补充哪些信息

## 有输入时
- 主题名 → 汇总该主题所有调研卡片，生成综述报告
- "结论" → 从已有信息中提炼核心结论 + 支撑证据
- "报告" → 生成完整调研报告（问题→方法→发现→结论→建议）
- 时间范围 → 总结该时段的调研进展

## 规则
- 结论必须有证据支撑，标注来源
- 区分事实（有来源）和推测（标记为 inference）
- 综述报告结构清晰：问题定义 → 调研方法 → 关键发现 → 对比分析 → 结论建议
- 如果信息不足以下结论，明确指出缺口而非强行总结
- 用 Mermaid 画概念关系图辅助理解
