收敛模式 — 检验掌握度，推荐今日复习，追踪进度。

## 分层 Review 汇报

输出按重要度分层，用户从上往下扫：

```
第1行: 一句话进度（10秒可读）
第2段: 今日推荐复习（间隔重复算法）
第3段: 薄弱点分析（需要关注的）
第4段: L0 汇总（自动消化了什么）
```

## 无输入时（融合 progress + review-cycle）

1. 扫描学习库所有概念笔记的 mastery 和 last_reviewed
2. 一句话总结："已掌握 N/M 概念，薄弱 X 个，未学 Y 个"
3. **今日推荐复习**（间隔重复算法）：
   - 选出 2-3 个到期需复习的概念
   - 优先级：red(薄弱) > green但到期 > gray(未测)
   - 间隔：1天→3天→7天→14天→30天
4. 薄弱点分析：连续答错的概念 + 建议强化方式
5. L0 汇总：最近自动消化了几篇素材、新增了几个概念
6. 建议下一步（1 件事）：
   - "做一轮 [[主题]] 测验"
   - "用费曼法解释 [[薄弱概念]]"
   - "🎉 全部掌握！要输出教程分享吗？"

## 有输入时

- 主题名 → 该主题掌握详情（表格：概念/mastery/上次复习/间隔天数）
- "这周" → 本周学习统计（新学N个/复习M个/掌握率变化）
- "准备考试" → 生成重点清单（薄弱+高频+易混淆 对比表）

## 规则

- 用数据说话（掌握率、间隔天数、趋势）
- 快速，一屏可扫完
- 像学习伙伴对话，不像考官
- 如果刚完成测验，续接结果而非重新扫描
