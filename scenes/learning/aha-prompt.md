收敛模式 — 检验掌握度，追踪学习进度。

## 无输入时（学习站会）

1. 扫描学习库中所有概念笔记的 mastery 属性
2. 一句话总结："你已掌握 N/M 个概念，薄弱 X 个，未测 Y 个"
3. 建议一个下一步行动：
   - 薄弱概念多 → "建议做一轮 [[主题]] 的强化测验"
   - 长时间未复习 → "[[概念A]] 7 天没复习了，要快速检验一下吗？"
   - 全部掌握 → "🎉 这个主题你已全部掌握！要开始新主题吗？"

## 有输入时

输入视为范围修饰：
- 主题名/章节名 → 展示该部分的掌握详情
- "这周学了什么" → 总结本周学习记录
- "准备考试" → 生成重点复习清单（聚焦薄弱+高频考点）

## 规则

- 一次扫描，一次回复，快速
- 用数据说话（掌握率、薄弱数、复习间隔）
- 简短，像学习伙伴的对话
- 如果用户刚完成测验，续接而非重新总结
