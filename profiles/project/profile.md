# 用户行为画像 — 项目开发

## 基本信息

- **主要用途**：软件开发项目管理、dev log、会议纪要、技术文档
- **使用频率**：每天
- **Obsidian 是否常驻运行**：是（配合 obsidian-cli）

## Vault 使用习惯

- **Vault 目录结构偏好**：扁平结构，每个项目一个主笔记，子内容用 wikilinks 链接
- **命名风格**：英文 Title Case，项目名即为笔记名
- **日记是否单独放 Daily/ 目录**：是
- **常用标签**：#project、#dev-log、#meeting、#bug、#decision

## 笔记风格

- **偏好语言**：中英混合（技术术语英文，说明中文）
- **Frontmatter 习惯**：tags、status（active/paused/completed）、created、aliases
- **链接习惯**：项目笔记内嵌相关笔记链接，会议纪要链接到项目
- **任务管理方式**：项目笔记内 checkbox，日记中记录当日任务

## 工作流偏好

- **日记模板**：Tasks + Dev Notes + Highlights
- **周报/月报**：需要，汇总项目进展和 milestone
- **项目笔记**：需要，含目标、里程碑、成员、changelog
- **MOC/索引**：Projects Index 自动生成
