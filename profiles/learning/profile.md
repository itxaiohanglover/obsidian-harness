# 用户行为画像 — 学习备考

## 基本信息

- **主要用途**：考试复习、技术学习、读书笔记、知识体系化
- **使用频率**：每天
- **Obsidian 是否常驻运行**：是

## Vault 使用习惯

- **Vault 目录结构偏好**：扁平结构，按学科/主题建 index 笔记
- **命名风格**：中文 Title Case
- **日记是否单独放 Daily/ 目录**：是
- **常用标签**：#study、#review、#mastered、#confused、#exam

## 笔记风格

- **偏好语言**：中文为主，技术术语英文
- **Frontmatter 习惯**：tags、mastery（0-5）、reviewed（日期）、aliases
- **链接习惯**：概念笔记互相链接，形成知识网络
- **任务管理方式**：日记中的学习计划 checkbox

## 工作流偏好

- **日记模板**：Learning Plan + Today's Focus + Review Queue + Notes
- **周报/月报**：需要，统计学习进度和掌握程度
- **项目笔记**：按考试/课程组织复习计划
- **MOC/索引**：按学科自动生成，标注掌握程度
